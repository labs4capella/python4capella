h1. Python4Capella project structure

The Python4Capella and Python4CapellaSampleScripts projects provide everything needed to interact with your Capella project from your Python scripts.

* *Python4Capella* project:
** *java_api*: This folder contains the low level APIs needed to interact with the Java world.
** *resources*: This folder contains sample resources for sample scripts.
** *simplified_api*: This folder contains the simplified APIs to interact with Capella and some of its add-ons.
** *utilies*: This folder contains utility script that helps for interactions with Eclipse.

* *Python4CapellaSampleScripts* project:
** *sample_scripts*: This folder contains some sample scripts that demonstrate how you can use the simplified APIs.
