Configure PyDev to remove false errors

This section explains how you can configure PyDev to remove false errors. This will help to Write your scripts.

You will need to configure PyDev preferences if it’s not already done:

  • select the Window > Preferences menu
  • with the filter a the top left corner search for "analysis"
  • select the Code Analysis node on the left
  • select the Undefined tab and add the following keywords:
    • include,loadModule,argv
  • click the Apply and Close button

When you load an EASE module and use specific methods from it, you can first declare the methods before the loadModule statement: 

def showInputDialog():
    pass

loadModule('/System/UI')

componentName = showInputDialog("Enter the component name:", "New Component 1")

This prevent the call to showInputDialog to be marked as an error by PyDev.
